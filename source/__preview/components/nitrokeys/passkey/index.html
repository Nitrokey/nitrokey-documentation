Nitrokey Passkey#

The Nitrokey Passkey is the successor to the Nitrokey FIDO2. It is build on top of the technologies and framework used within the Nitrokey 3. Find the latest release notes on GitHub.

Please see the FIDO2 pages for more FIDO2 related documentation.

Currently there is only one firmware version, so no firmware update is necessary.
